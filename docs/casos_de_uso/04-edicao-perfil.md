# Edição de Perfil

## Descrição
Permite que o usuário atualize seus dados pessoais no sistema.

---

## User Story

Como usuário autenticado  
Quero editar minhas informações  
Para manter meus dados atualizados.

---

## Caso de Uso

**Nome:** Editar Perfil  
**Ator:** Usuário

### Fluxo Principal

1. O usuário acessa a tela de perfil.
2. O usuário altera os dados desejados.
3. O sistema salva as alterações.

---

## Critérios de Aceitação

- O usuário pode alterar nome.
- O usuário pode alterar foto de perfil.
- O sistema deve salvar as alterações.
