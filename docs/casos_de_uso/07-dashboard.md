# Dashboard de Produção

## Descrição
Apresenta uma visão geral da produção monitorada pelas máquinas.

---

## User Story

Como usuário do sistema  
Quero visualizar um dashboard de produção  
Para acompanhar o desempenho das máquinas.

---

## Caso de Uso

**Nome:** Visualizar Dashboard  
**Ator:** Usuário

### Fluxo Principal

1. O usuário realiza login.
2. O sistema carrega dados de produção.
3. O sistema exibe gráficos e indicadores.

---

## Critérios de Aceitação

- O dashboard deve mostrar dados de produção.
- O dashboard deve atualizar automaticamente.
